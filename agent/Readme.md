# Backend (API)

## Description

- Agent 구현을 위한 서버

## Environment

- Python 3.11.2
- FastAPI 0.136.1

## Getting Started

```bash
uvicorn app.main:app --port 8000
```

http://localhost:8000 에서 확인 가능

## Etc

- 파이썬 가상환경 세팅 후 실행
- requirements.txt 의존성 확인 후 설치 후 실행
- AI Agent 위한 환경변수 세팅 후 실행
- /docs 경로에서 swagger 문서 확인 가능
