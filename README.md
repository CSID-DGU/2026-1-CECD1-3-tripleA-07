## Main Feature

### Web & API

- Travel Product Management
  - Retrieve a travel product
  - Search travel products
  - Create a travel product
  - Update a travel product
- SNS Advertisement History Management
  - Search SNS advertisement publishing records

### Agent

- Marketing Agent
  - Few-shot and RAG-based prompting
- Review Agent
  - Chain-of-Thought (CoT) and Self-Consistency-based prompting
